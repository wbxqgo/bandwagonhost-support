## 搬瓦工工单提交教程（一步一步来）

搬瓦工的工单系统在账号后台，英文界面，但操作不复杂。

1. **登录搬瓦工客户中心**：打开搬瓦工官网，点击右上角 "Client Area" 进入账号后台。
2. **找到工单入口**：在顶部导航栏找到 "Support" → 点击 "Tickets"，这就是工单中心。
3. **新建工单**：点击 "Open New Ticket"，开始填写。
4. **填写工单内容**：
   - **Subject**：工单主题，简短描述问题（如 "My IP is blocked"）
   - **Related Service**：选择对应的 VPS 服务；没有关联服务选 "None"
   - **Message**：详细描述问题，包括报错信息、发生时间、你已经尝试的解决方法
   - **Attachments**：可以附上截图、日志文件（非必须）
5. **提交工单**：点击 "Submit"，等待回复。

工单内容必须用英文写。不会英文直接把中文描述丢进翻译软件，翻出来的内容客服基本都能看懂。

## 回复是机器人怎么办？

这是很多人遇到过的情况。提交完工单，过了半小时，回复来了，署名是 **Martin C**，然后里面写着一句话：

> *Note: I am a robot and my reply is based on the information provided in the ticket, as well as diagnostics data I have access to.*

这就是搬瓦工的自动机器人回复。它对常见问题有预设答案，回复速度快但未必解决了你的问题。

遇到这种情况，**直接在同一个工单下继续回复**，把你的问题再描述一遍。这样会自动进入人工客服处理流程。不需要重新开新工单。

通常再等几小时，人工客服就会接手。

## 客服邮件怎么写才有回音

如果你用的是邮件联系（support@bandwagonhost.com），写法很关键。客服是纯英文环境，邮件需要包含以下几个基本要素：

- **你的账号邮箱**（搬瓦工注册时用的那个）
- **服务 ID 或订单号**（在后台 My Products 里可以查到）
- **问题描述**：发生了什么、什么时候开始的、你做了什么操作
- **期望结果**：你希望客服帮你做什么

邮件标题直接写问题类型，比如 "Request for IP replacement - Service #XXXXX"，比写 "Help me!!!" 容易被处理。
